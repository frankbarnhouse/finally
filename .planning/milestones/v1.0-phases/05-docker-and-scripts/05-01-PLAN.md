1. Add `"litellm>=1.80.0"` to the `dependencies` list in `backend/pyproject.toml`. This dependency is used by `backend/app/services/chat.py` but was never added to pyproject.toml (it was pip-installed manually). Without it, `uv sync` in Docker will fail to install litellm.

2. Run `cd backend && uv lock` to regenerate the lockfile with litellm included.

3. In `backend/app/main.py`, add conditional static file serving AFTER all router includes. The static files directory only exists in Docker (copied from frontend build). Use pathlib to check if the directory exists before mounting:

```python
from pathlib import Path

# After all app.include_router() calls:
static_dir = Path(__file__).resolve().parent.parent / "static"
if static_dir.is_dir():
    app.mount("/", StaticFiles(directory=str(static_dir), html=True), name="static")
```

This mounts the static export at "/" with `html=True` so that `index.html` is served for "/". The `if` guard means it only activates when the static/ directory exists (i.e., in Docker), so local dev is unaffected. Remove the `# noqa: F401` comment from the StaticFiles import since it is now used.

3. Create `Dockerfile` at project root. Multi-stage build:

**Stage 1 (frontend-build):** Use `node:22-slim` (latest LTS). Copy `frontend/` directory. Run `npm ci` then `npm run build`. Output goes to `frontend/out/`.

**Stage 2 (production):** Use `python:3.12-slim`.
- Install uv: `COPY --from=ghcr.io/astral-sh/uv:latest /uv /uvx /bin/`
- Set workdir to `/app`
- Copy `backend/pyproject.toml` and `backend/uv.lock` first (layer caching)
- Run `uv sync --frozen --no-dev` to install production deps only
- Copy `backend/app/` to `/app/app/`
- Copy frontend build: `COPY --from=frontend-build /app/frontend/out /app/static`
- Create db directory: `RUN mkdir -p /app/db`
- Expose 8000
- ENV `PYTHONUNBUFFERED=1`
- CMD: `["/app/.venv/bin/uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]`

Key details:
- Use `npm ci` (not `npm install`) for reproducible builds
- Use `uv sync --frozen --no-dev` so it uses the lockfile exactly and skips dev deps
- The `--frozen` flag means uv will not update the lockfile, just install from it
- Copy pyproject.toml and uv.lock before app code for Docker layer caching
- Static files land at `/app/static/` which matches the `static_dir` path in main.py (relative to main.py at `/app/app/main.py`, parent.parent = `/app/`)
