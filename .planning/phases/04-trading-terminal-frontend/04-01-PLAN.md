SSE endpoint: GET /api/stream/prices

```json
{"AAPL": {"ticker": "AAPL", "price": 190.50, "previous_price": 190.25, "timestamp": 1710600000.0, "change": 0.25, "change_percent": 0.1314, "direction": "up"}, ...}
```

GET /api/portfolio returns:
```json
{"positions": [{"ticker": "AAPL", "quantity": 10, "avg_cost": 185.00, "current_price": 190.50, "unrealized_pnl": 55.00}], "cash_balance": 8150.00, "total_value": 10055.00}
```

GET /api/watchlist returns:
```json
[{"ticker": "AAPL", "price": 190.50}, ...]
```

POST /api/portfolio/trade — request: {ticker, quantity, side}
Success: {trade: {...}, cash_balance, position}
Error: {error: "..."}

GET /api/portfolio/history returns:
```json
[{"total_value": 10000.00, "recorded_at": "2026-03-16T12:00:00+00:00"}, ...]
```

POST /api/chat — request: {message}
Response: {message, actions: {trades: [...], watchlist_changes: [...]}}

1. Create `src/types/index.ts` with these exact interfaces:
   - `PriceUpdate`: ticker (string), price (number), previous_price (number), timestamp (number), change (number), change_percent (number), direction ('up' | 'down' | 'flat')
   - `Position`: ticker (string), quantity (number), avg_cost (number), current_price (number), unrealized_pnl (number)
   - `Portfolio`: positions (Position[]), cash_balance (number), total_value (number)
   - `TradeRequest`: ticker (string), quantity (number), side ('buy' | 'sell')
   - `TradeResult`: trade (object with ticker, side, quantity, price, executed_at), cash_balance (number), position (object with ticker, quantity, avg_cost OR null)
   - `ChatResponse`: message (string), actions (object with trades array and watchlist_changes array)
   - `WatchlistItem`: ticker (string), price (number | null)

2. Create `src/lib/api.ts` with fetch wrappers (all same-origin `/api/*`):
   - `fetchPortfolio(): Promise<Portfolio>` — GET /api/portfolio
   - `executeTrade(req: TradeRequest): Promise<TradeResult>` — POST /api/portfolio/trade, throws on error response
   - `fetchPortfolioHistory(): Promise<{total_value: number, recorded_at: string}[]>` — GET /api/portfolio/history
   - `fetchWatchlist(): Promise<WatchlistItem[]>` — GET /api/watchlist
   - `addToWatchlist(ticker: string): Promise<void>` — POST /api/watchlist with {ticker}
   - `removeFromWatchlist(ticker: string): Promise<void>` — DELETE /api/watchlist/{ticker}
   - `sendChatMessage(message: string): Promise<ChatResponse>` — POST /api/chat with {message}
   All functions use native fetch. On non-ok response, read JSON and throw error message.

3. Create `src/stores/priceStore.ts` — Zustand store:
   - State: `prices: Record<string, PriceUpdate>`, `priceHistory: Record<string, {time: number, value: number}[]>`, `connectionStatus: 'connected' | 'connecting' | 'disconnected'`
   - `updatePrices(data: Record<string, PriceUpdate>)` — merges into prices, appends to priceHistory (cap at 500 entries per ticker)
   - `setConnectionStatus(status)` — updates connection status

4. Create `src/stores/portfolioStore.ts` — Zustand store:
   - State: `positions: Position[]`, `cashBalance: number`, `totalValue: number`, `loading: boolean`
   - `fetchPortfolio()` — calls api.fetchPortfolio(), updates state
   - `executeTrade(req: TradeRequest)` — calls api.executeTrade(), then re-fetches portfolio

5. Create `src/stores/chatStore.ts` — Zustand store:
   - State: `messages: {role: 'user' | 'assistant', content: string, actions?: ChatResponse['actions']}[]`, `loading: boolean`
   - `sendMessage(message: string)` — adds user message, sets loading, calls api.sendChatMessage(), adds assistant response with actions, clears loading

6. Create `src/hooks/useSSE.ts`:
   - Uses `useEffect` to create `new EventSource('/api/stream/prices')`
   - `onopen` -> setConnectionStatus('connected')
   - `onmessage` -> parse JSON, call updatePrices
   - `onerror` -> setConnectionStatus('disconnected')
   - Cleanup: `es.close()`
   - Subscribe to Zustand actions via selectors to avoid re-render loops

7. Create `src/hooks/useLightweightChart.ts`:
   - Accepts containerRef and optional chart options
   - Creates chart with dark theme: background #0d1117, text #8b949e, grid lines #21262d
   - Returns chartRef and seriesRef
   - Handles resize via ResizeObserver (NOT window resize — more reliable for panel layouts)
   - Cleanup: remove chart on unmount

8. Create `src/components/ConnectionDot.tsx`:
   - Reads `connectionStatus` from priceStore via selector
   - Renders a small colored circle: green (#27ae60) for connected, yellow (#ecad0a) for connecting, red (#e74c3c) for disconnected
   - Include a title attribute with the status text

9. Create `src/components/Header.tsx`:
   - "use client" component
   - Shows app name "FinAlly" on the left
   - Shows portfolio total value (formatted as currency) and cash balance from portfolioStore
   - Shows ConnectionDot on the right
   - Styled with bg-terminal-surface, border-b border-terminal-border
   - Calls `portfolioStore.fetchPortfolio()` in a useEffect on mount

10. Create `src/app/page.tsx`:
    - "use client" component
    - Calls `useSSE()` to start the SSE connection
    - Renders a grid layout:
      - Header at top (full width)
      - Left column (~70%): placeholder divs for Watchlist, MainChart, Portfolio sections
      - Right column (~30%): placeholder div for ChatPanel
    - Each placeholder has bg-terminal-surface border border-terminal-border rounded with a muted text label
    - Grid uses CSS grid with `h-screen` and `grid-rows-[auto_1fr]`
